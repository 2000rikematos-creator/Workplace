# 🛠️ Workplace

Workplace is an app for managing a company's staff 

---

## 🚀 Technologies Used 

### Frontend
- **React** (with TypeScript)
- **Vite** (build and development)
- **CSS3** (styling)

### Backend
- **Node.js** & **Express** (with TypeScript)
- **PostgreSQL** (hosted via supabase)

---

## 📦 How to install and run locally

- 1. Clone the repository

- 2. Go to /Backend and run npm install

- 3. Go to /Frontend and run npm install

- 4. Go to to both directories and change the name of the file .env.example to .env in both directories and add your variables there

- 5. Go to /Backend/schema.sql and copy the queries to your SQL editor

- 6. Go to /Backend and run npm run build and then npm start

- 7. Go to /Frontend and run npm run build and then npm run preview
